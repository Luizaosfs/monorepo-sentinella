## COMO PREENCHER ESTA LISTA

### Fontes para identificar municípios
- **Situação da dengue:** Boletim epidemiológico estadual (SESA) | SINAN online (datasus.gov.br) | Infodengue (info.dengue.mat.br)
- **Secretário de saúde:** Site da prefeitura | Portal da Transparência | LinkedIn | Ligação direta para a secretaria
- **Agentes de endemias:** Contato com a Vigilância Epidemiológica municipal | CNES (datasus)
- **Possui sistema:** Perguntar diretamente | Buscar licitações abertas no portal de transparência

### Critérios de priorização (ordenar a lista por pontuação)

| Critério | Peso | Como avaliar |
|---|---|---|
| Dengue endêmica há ≥ 3 anos | 3 pts | SINAN: casos confirmados todo ano |
| Não possui sistema | 3 pts | Confirmado por contato direto |
| Secretário acessível / responsivo | 2 pts | Atende WhatsApp/e-mail em < 48h |
| Município próximo geograficamente | 2 pts | < 3h do responsável Sentinella |
| Equipe ≥ 5 agentes de campo | 1 pt | CNES ou informação direta |
| **Pontuação máxima** | **11 pts** | |

## REFERÊNCIAS PARA PESQUISA

### Dados de dengue por município
- **SINAN:** `http://sinan.saude.gov.br/sinan/login/login.jsf`
- **Infodengue:** `https://info.dengue.mat.br`
- **SESA do estado:** buscar "boletim epidemiológico dengue [estado] 2025"

### Dados de municípios
- **IBGE Cidades:** `https://cidades.ibge.gov.br`
- **Portal da Transparência (secretários):** buscar "secretário de saúde [município]"
- **CNES (agentes e unidades):** `http://cnes.datasus.gov.br`

### Identificar municípios sem sistema
- Buscar no portal de compras municipais: "contratos de software vigilância epidemiológica"
- Ligar para a Vigilância Epidemiológica e perguntar: "vocês usam algum sistema para registro de vistorias?"
