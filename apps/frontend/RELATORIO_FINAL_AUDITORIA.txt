DESCOBERTAS PRINCIPAIS
================================================================================

1. ARQUITETURA
   ├─ Hook simples usando Context React
   ├─ localStorage para persistência
   ├─ Provider em nível raiz (App.tsx)
   └─ 4 consumidores do estado

2. CHAVE LOCALSTORAGE
   ├─ Nome: 'sentinella_modo_analitico'
   ├─ Valores: '1' (ativo) ou '0' (inativo)
   ├─ Falha segura: padrão false
   └─ Ignora erros de quota/modo privado

3. TOGGLE UI
   ├─ Localização: Sidebar footer (AppLayout:722-740)
   ├─ Ícone: FlaskConical
   ├─ Labels: "Modo analítico" / "Analítico ativo"
   ├─ Cores: branco/cinza (inativo) → azul (ativo)
   └─ Sempre visível se autorizado

4. ACESSO
   ├─ Autorizado:
   │  ├─ admin (via isAdminOrSupervisor)
   │  ├─ supervisor (via isAdminOrSupervisor)
   │  └─ analista_regional (papel === 'analista_regional')
   └─ Bloqueado: agente, notificador

5. EFEITO
   ├─ Mostra/esconde bloco "Análise detalhada"
   ├─ Em FichaImovel360.tsx (linha 384)
   ├─ Em GestorFocoDetalhe.tsx (linha 653)
   ├─ Condicional: modoAnalitico && v.prioridade_final
   └─ Apenas alteração visual, não calcula dados

6. COMPONENTE DE RENDERIZAÇÃO
   ├─ Nome: ConsolidacaoAnaliticaDetalhe.tsx
   ├─ Bloco azul claro com border azul
   ├─ Exibe 5 dimensões:
   │  ├─ Resultado operacional (3 valores)
   │  ├─ Vulnerabilidade domiciliar (5 valores)
   │  ├─ Alerta de saúde (4 valores)
   │  ├─ Risco socioambiental (4 valores)
   │  └─ Risco vetorial (5 valores)
   ├─ Cores por severidade: verde → amarelo → laranja → vermelho
   └─ NULL: "Não coletado" (cinza)

7. DASHBOARD ANALÍTICO
   ├─ Rota: /gestor/dashboard-analitico
   ├─ Arquivo: src/pages/gestor/DashboardAnalitico.tsx
   ├─ Guard: DashboardAnaliticoGuard
   ├─ Acesso: admin, supervisor, analista_regional
   ├─ É COMPLETAMENTE INDEPENDENTE do toggle
   ├─ O toggle não controla acesso ao Dashboard
   └─ Dashboard não afeta o toggle

8. ESTADO GLOBAL
   ├─ Leitura: useModoAnalitico().ativo
   ├─ Escrita: useModoAnalitico().toggle()
   ├─ Persistente: sobrevive refresh
   └─ Simples e manutenível
