# Fluxo de Operação Manual

## Objetivo
Permitir que operadores realizem inspeções manuais em campo.

## Etapas

### 1 Planejamento
Um planejamento manual é criado para uma região ou bairro.

### 2 Execução da vistoria
O operador realiza a inspeção presencialmente.

Durante a vistoria o operador pode:

- registrar problemas
- capturar fotos
- marcar localização

### 3 Registro de evidência
Cada problema identificado gera um levantamento_item.

As evidências podem incluir:

- fotos
- localização GPS
- descrição do problema

### 4 Classificação
O operador classifica o problema quanto a:

- tipo
- risco
- prioridade

### 5 Plano de ação
Após a identificação do problema, o operador pode registrar o plano de ação.

Exemplos:

- eliminar água parada
- notificar responsável
- aplicar tratamento
- solicitar limpeza

### 6 Conclusão
O item pode ser marcado como resolvido ou pendente.
